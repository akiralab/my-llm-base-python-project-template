# Code of Conduct

## 1. 基本姿勢

- 相手の背景や経験を尊重し、建設的に対話する。
- 技術的な指摘は、事実と再現可能な根拠に基づいて行う。
- 個人攻撃・侮辱・威圧的行為を禁止する。

## 2. コラボレーション原則

- 仕様・前提・制約を明示して議論する。
- レビューでは改善提案を添える。
- AI生成物の扱いでも、人間同士の敬意を維持する。

## 3. 禁止事項

- ハラスメント、差別、脅迫
- 機密情報の無断共有
- 意図的な品質低下を招く行為

## 4. 問題発生時の対応

- まずは担当Maintainerへ連絡する。
- 重大な違反は管理者が調査し、必要に応じてアクセス制限を行う。
